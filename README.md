# Football Training Planner

## Project Description
Football Training Planner is a ReactJS frontend web application designed to help football players organize their weekly training schedule and follow simple nutrition tips. The project includes multiple pages, a sidebar navigation layout, and a contact form.

## Pages
- Home
- About
- Training
- Nutrition
- Contact

## Technologies Used
- ReactJS
- React Router DOM
- HTML
- CSS
- Git
- GitHub

## Features
- Responsive dashboard layout
- Left sidebar navigation
- Multiple pages using React Router
- Training plan page
- Nutrition tips page
- Contact form using React useState
- Clean UI/UX design
